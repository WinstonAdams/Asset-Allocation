# SC-048: 行為協定頁唯讀渲染協定全文

## 行為
- **GIVEN** 行為協定全文文件存在於系統可存取位置
- **WHEN** 使用者開啟「行為協定」頁
- **THEN**
  - 以 markdown 唯讀方式呈現協定全文（含各節：存在理由、情境分級、機動加碼規則、行為防火牆、事前授權例外、檢核）
  - 顯示的是文件當前最新內容
  - 頁面無任何編輯／儲存入口（純唯讀）

## 錯誤
- 協定文件缺失／無法讀取時：顯示友善錯誤提示，不顯示技術堆疊、不使整頁崩壞
