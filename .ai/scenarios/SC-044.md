# SC-044: 回撤深度對應 L0/L1/L2/L3 四級，達門檻即進入較深一級

## 行為
- **GIVEN** 資料充足（累積 TWR 有效月數 ≥ 3），且有效門檻為預設值——自高點回撤達 10% 進 L1、20% 進 L2、30% 進 L3
- **WHEN** 系統依「目前自歷史高點回撤深度 d」判定協定等級
- **THEN** 依回撤深度落入四級之一：
  - d < 10% → L0（平時）
  - 10% ≤ d < 20% → L1（修正）
  - 20% ≤ d < 30% → L2（熊市）
  - d ≥ 30% → L3（深熊）

## 邊界
達門檻即進入較深一級（回撤深度恰等於門檻時歸入較深級）：
- d 恰為 10% → L1（不是 L0）
- d 恰為 20% → L2（不是 L1）
- d 恰為 30% → L3（不是 L2）
- d = 0%（無回撤／正處新高）→ L0
- d = 9.99% → L0；d = 19.99% → L1；d = 29.99% → L2
