# SC-014: 建倉月期初市值為0不納入TWR連乘

## 行為
- **GIVEN** 某資產項目於首次建倉月期初市值為 0（該月才首次投入）
- **WHEN** 系統計算 TWR
- **THEN** 期初市值為 0 的建倉月不納入連乘（避免除以 0）
- **AND** TWR 連乘自下一個期初市值大於 0 的期間起算
