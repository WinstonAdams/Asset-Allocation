# SC-043: 回撤基準以累積 TWR 成長指數計算，歷史高點納入起始建倉基準

## 行為
- **GIVEN** 投資組合已有逐月累積 TWR 序列（僅計資產、已排除當月淨投入），例如三個有資料月的累積 TWR 依序為 +5%、−8%、−22%
- **WHEN** 系統計算「目前自歷史高點的回撤」
- **THEN**
  - 以累積 TWR 建立成長指數路徑，並在路徑最前端納入建倉基準點（起始月 0% 累積報酬，指數 = 1.0）
  - 歷史高點取整條指數路徑（含起始基準 1.0）的最大值
  - 目前回撤 = 最新一點相對歷史高點的跌幅（一個 ≤ 0 的百分比）
  - 以上例：指數路徑 = 1.00、1.05、0.92、0.78；歷史高點 = 1.05；目前回撤 = 0.78 / 1.05 − 1 ≈ −25.7%

## 邊界
- 只跌不漲的投組（每月累積 TWR 皆為負，如 −6%、−13%、−19%）：歷史高點仍以起始基準 1.0 認定（而非以「第一個已下跌月」當高點），以免低報跌幅。上例指數路徑 = 1.00、0.94、0.87、0.81，歷史高點 = 1.00，目前回撤 = 0.81 / 1.00 − 1 = −19%（若不納入起始基準 1.0，會誤把高點當成 0.94、回撤僅約 −13.8%，低報跌幅）
- 投組正處歷史新高（最新累積 TWR 為序列最大且為正）：目前回撤 = 0%（最新點即歷史高點）
- 歷史高點出現在序列中段、之後回落：以中段高點量測目前回撤，而非以起始基準或最新點
