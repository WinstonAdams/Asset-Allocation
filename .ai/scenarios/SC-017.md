# SC-017: 賺賠金額與簡單總報酬率以累積成本起算

## 行為
- **GIVEN** 某資產項目初始成本 300000
- **AND** 後續各月淨投入合計 +100000
- **AND** 當前市值 500000
- **WHEN** 系統計算賺賠金額與簡單總報酬率
- **THEN** 累積成本 = 初始成本 300000 + 後續淨投入 100000 = 400000
- **AND** 賺賠金額 = 市值 500000 − 累積成本 400000 = 100000
- **AND** 簡單總報酬率 = 賺賠 100000 ÷ 累積成本 400000 = 25%

## 邊界
- 累積成本為 0 時，簡單總報酬率不顯示
