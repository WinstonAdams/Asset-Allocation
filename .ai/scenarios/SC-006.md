# SC-006: 新增/編輯/刪除某月單一項目紀錄

## 行為
- **GIVEN** 使用者在 2026-05 月度表單中操作某資產項目
- **WHEN** 新增一列：填入市值 520000、淨投入 0
- **THEN** 該 (2026-05, 項目) 紀錄被儲存
- **WHEN** 編輯該列市值改為 530000
- **THEN** 同一 (2026-05, 項目) 紀錄被更新為 530000，不新增重複列
- **WHEN** 刪除該列
- **THEN** 該 (2026-05, 項目) 紀錄被移除
