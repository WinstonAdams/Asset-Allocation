# SC-045: 累積 TWR 有效月數不足 3 個月時退回 L0 並顯示資料不足，不誤報大跌

## 行為
- **GIVEN** 累積 TWR 有效月數少於 3（定案：有效月數 < 3 一律視為資料不足）
- **WHEN** 系統判定協定等級
- **THEN**
  - 一律退回 L0（平時姿態）
  - 不觸發任何大跌等級（L1–L3）
  - 不顯示回撤深度數值、不顯示紅色警示
  - 依「無紀錄」與「有資料但不足」分兩種中性文案呈現（見邊界），避免以少數月雜訊誤報大跌

## 邊界
資料量與對應呈現（皆退回 L0、皆不顯示回撤數值）：
- 有效月數 = 0（完全無紀錄／尚無任何有效累積 TWR 節點）→ 提示「尚無資料，請先至月度錄入輸入」
- 有效月數 = 1 或 2（有資料但不足 3 個月）→ 提示「資料尚不足，暫不評估大跌等級」
- 有效月數 = 2 → 仍屬資料不足，沿用「資料尚不足」文案，退回 L0
- 有效月數 = 3 → 達下限，正式依回撤深度判定 L0–L3（不再視為資料不足）
