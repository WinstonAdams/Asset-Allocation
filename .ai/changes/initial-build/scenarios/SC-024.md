# SC-024: 某月未更新市值沿用上一個有值月份

## 行為
- **GIVEN** 某項目 2026-04 市值為 250000
- **AND** 2026-05 該項目仍在當月清單中，但市值欄留空（仍持有、當月未更新，如保險未取得當月解約金）
- **WHEN** 系統處理 2026-05 該項目市值
- **THEN** 沿用上一個有值月份（2026-04）的市值 250000
- **AND** 該沿用不被視為當月有變動

## 邊界
- 觸發條件＝「項目仍在當月清單中、市值欄留空」；規則適用所有資產，典型情境為保險解約金
- 三態互斥不重疊（與 SC-009、SC-022 邊界對齊）：
  - **市值留空（本卡）**＝仍持有、當月未更新 → 沿用上一有值月份市值（BR-10b）
  - **把項目從當月移除、不建該項目列**＝已賣出 → 之後缺列，承 SC-009（賣出當月先記市值 0＋負淨投入，再省略後續列）
  - **整個月份完全沒有任何項目紀錄**＝缺月 → 報酬以相鄰有資料月分段連乘、不補插，承 SC-022
- 「市值留空沿用」產生的是該項目該月的有值節點（沿用值），不是缺月；故不觸發 SC-022 的分段跳過
