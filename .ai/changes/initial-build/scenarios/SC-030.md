# SC-030: 偏離超過門檻標示需再平衡

## 行為
- **GIVEN** 再平衡偏離門檻設為 5 百分點
- **AND** 某分類偏離為 +8 百分點
- **WHEN** 系統判定是否需再平衡
- **THEN** 該分類偏離絕對值 8 > 門檻 5，標示「需再平衡」
- **GIVEN** 另一分類偏離為 +3 百分點
- **THEN** 偏離絕對值 3 ≤ 門檻 5，不標示再平衡

## 邊界
- 偏離絕對值剛好等於門檻（5 = 5）不標示再平衡
- 未設定目標比重的分類不顯示偏離、不判定再平衡
