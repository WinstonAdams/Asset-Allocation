# SC-037: 項目間轉移的輸入防呆

## 行為
- **GIVEN** 使用者要在某月份做「項目間轉移」
- **WHEN** 轉移金額不是正數（為 0 或負數），或來源項目與目標項目是同一個
- **THEN** 系統拒絕該次轉移，回報輸入無效，且不產生任何月度紀錄

## 邊界
- 金額為 0：沒有任何資金移動，視為無效輸入，拒絕
- 金額為負：方向語意與「正金額由來源轉向目標」相反、易誤導，視為無效輸入，拒絕
- 來源等於目標：同一項目自己轉給自己、淨效果為 0、形同未轉移，視為無效輸入，拒絕
- 金額為正且來源與目標不同：通過防呆，照常成對記錄（來源 −amount、目標 +amount）

## 備註
- 正常路徑（正金額、來源與目標不同）已由 SC-008 涵蓋；本卡聚焦無效輸入的拒絕。
